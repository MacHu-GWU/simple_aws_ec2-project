Install
------------------------------------------------------------------------------

``simple_aws_ec2`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install simple_aws_ec2

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade simple_aws_ec2
